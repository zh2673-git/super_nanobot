## 一、核心类解析

### 1.1 AgenticSearch 类

**文件位置**：`src/sirchmunk/search.py`

**类比解释**：AgenticSearch 就像一个**餐厅的店长**，负责统筹整个搜索流程：
- 接收顾客的点餐（用户查询）
- 分析顾客想要什么（查询意图分析）
- 安排不同部门工作（调度各个模块）
- 把最终结果端给顾客（返回搜索结果）

#### 1.1.1 类结构

```python
class AgenticSearch(BaseSearch):
    """智能搜索核心类"""
    
    def __init__(
        self,
        llm: Optional[OpenAIChat] = None,
        work_path: Optional[Union[str, Path]] = None,
        paths: Optional[Union[str, Path, List[str], List[Path]]] = None,
        verbose: bool = True,
        log_callback: LogCallback = None,
        reuse_knowledge: bool = True,
        **kwargs,
    ):
        # 初始化搜索系统
```

#### 1.1.2 核心方法

| 方法 | 功能 | 类比 |
|------|------|------|
| `__init__` | 初始化搜索系统 | 店长准备开店 |
| `search()` | 执行搜索 | 店长接待顾客并安排工作 |
| `_analyze_query()` | 分析查询意图 | 店长询问顾客想吃什么 |
| `_extract_keywords()` | 提取关键词 | 店长记录顾客的需求要点 |
| `_retrieve_text()` | 检索文本 | 服务员去厨房拿食材 |
| `_query_knowledge()` | 查询知识库 | 查菜单看有没有这道菜 |
| `_process_evidence()` | 处理证据 | 厨师准备食材 |
| `_summarize_results()` | 总结结果 | 摆盘上菜 |

#### 1.1.3 search() 方法详解

```python
async def search(
    self,
    query: str,
    mode: str = "search",
    k: int = 10,
    paths: Optional[List[str]] = None,
    # ... 其他参数
) -> SearchResponse:
    """
    执行搜索的主方法
    
    流程：
    1. 判断是否为闲聊查询
    2. 提取关键词
    3. 检索文本
    4. 查询知识库
    5. 处理证据
    6. LLM 总结
    7. 返回结果
    """
```

    """
    搜索接口
    
    参数：
    - query: 搜索查询
    - mode: 搜索模式 (search/chat)
    - k: 返回结果数量
    
    返回：
    - SearchResponse: 搜索结果
    """
    # 1. 创建 AgenticSearch 实例
    search_engine = AgenticSearch(...)

## 五、LLM 集成

### 5.1 OpenAIChat 类

**文件位置**：`src/sirchmunk/llm/openai_chat.py`

**类比解释**：OpenAIChat 就像一个**翻译员**，负责：
- 把用户的问题"翻译"成 LLM 能理解的格式
- 把 LLM 的回答"翻译"成用户能看懂的格式

#### 5.1.1 核心方法

```python
class OpenAIChat:
    """OpenAI 聊天客户端"""
    
    async def chat(
        self,
        messages: List[Message],
        system_prompt: str = None,
        temperature: float = 0.7,
        # ... 其他参数
    ) -> str:
        """
        发送聊天请求
        
        参数：
        - messages: 消息列表
        - system_prompt: 系统提示词
        - temperature: 创造性参数
        
        返回：
        - LLM 生成的回复文本
        """
        # 1. 构造请求
        payload = {
            "model": self.model,
            "messages": messages,
            "temperature": temperature,
        }
        
        # 2. 调用 OpenAI API
        response = await self.client.chat.completions.create(**payload)
        
        # 3. 返回结果
        return response.choices[0].message.content
```
